MASTER MART REGISTRY — 7 FJELL BRYGGERI BI SUITE
Schema: "7fsalg"
Last updated: April 2026

--------------------------------------------------
HOW TO USE THIS DOCUMENT
--------------------------------------------------

This registry tracks build status, dependencies, and dashboard assignments
for every mart in the BI suite. Column definitions live in schema.txt.

Build order:
1. Foundation marts
2. Analytical time-series marts
3. Portfolio structure marts
4. Index normalisation marts
5. Customer analytical marts
6. SKU analytical marts
7. Pricing & margin marts
8. Channel-specific marts
9. Dashboard-specific marts

No mart is built without a registry entry.
No mart is built before its dependencies are complete.

INDEX NORMALISATION MARTS
--------------------------------------------------

Index formula: ROUND(volume_litres / base_period_volume * 100, 1)
Base period: first month where volume_litres > 0 per dimension member.
Never reimplement index logic in other marts or in Metabase.
